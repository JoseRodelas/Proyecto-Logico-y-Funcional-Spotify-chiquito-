Acerca de nosotros...

Somos estudiantes de la carrera de Ingeniería en sistemas computacionales, cursando el 8 y penúltimo semestre de dicha carrera académica; y este trabajo centrado en el uso de CSS mediante figma en complemento con Bootstrap para crear una pagina responsiva, también llevando a cabo la edición del proyecto de manera grupal y el manejo de versiones mediante github, con el uso del lenguaje de programacion llamado Angular, es de la materia de Programación lógica y funcional, para llevar a la práctica de los conocimientos obtenidos en dicha materia.

Este trabajo se centra en la recreación de una versión simplificada de la plataforma Spotify, utilizando Angular como framework principal para el desarrollo frontend. El objetivo es replicar, de forma lógica y funcional, algunas de las secciones más representativas de la interfaz de usuario de Spotify, integrando elementos de diseño modernos y componentes reutilizables para garantizar una navegación fluida y coherente.
